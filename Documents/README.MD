# PowerOR Project Documents

This folder contains the topic-specific documentation for the PowerOR project.

PowerOR is an experimental PlayStation 2 Slim power OR-ing project. The goal is to explore the use of USB-C PD as an optional power source while keeping the original OEM PS2 power supply as the preferred/default input.

This documentation is organized to follow the project from basic theory, through design discussion, testing, fitment, and eventual kit planning.
